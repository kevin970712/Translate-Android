# Translate

**Translate** is a lightweight Google Translate alternative.

## Features

*   **Lightweight:** The app size is under 10MB.
*   **Privacy-Friendly:** Use simplytranslate as the backend.
*   **Clean UI:** A simple interface that anyone can easily use.
